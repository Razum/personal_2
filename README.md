# razumieienko.com

Personal portfolio website for Roman Razumieienko — UI/UX Lead Developer with 15+ years of experience building polished digital experiences across healthcare, fintech, entertainment, gaming, telecom, and AI.

## Sections

- **Hero** — animated canvas background with introductory tagline
- **About** — bio, portrait, and key career statistics
- **Experience** — timeline of professional roles and accomplishments
- **Skills** — categorized technical and leadership skill set
- **Contact** — email and social links (LinkedIn, GitHub, Wellfound, Xing)

## Tech stack

- [Astro](https://astro.build/) v6
- TypeScript
- Vanilla CSS with custom properties and BEM naming
- Google Fonts (DM Sans + Playfair Display)

## Getting started

```sh
pnpm install
pnpm dev        # → localhost:4321
```

## Build

```sh
pnpm build      # production output → ./dist/
pnpm preview    # preview the build locally
```

## Requirements

- Node ≥ 22.12.0
- pnpm
